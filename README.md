## Linux
linux二进制启动器支持
### 介绍
它会自动下载最新的皎月连二进制文件并且记录版本号，可以打开时检查更新
提供更方便的交互选项，减少敲命令的过程
启动器依赖init建立服务文件  
#### 目前支持  
* systemd
* openrc  

欢迎其他有兴趣的贡献者提交对其他类型的init的支持

#### 使用
下载***natpierce_linux***包,然后运行里面的***gui.sh***脚本，剩下的按提示来
